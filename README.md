Используйте датасет data_hw_bank_customer

Вычислите статистики для возраста клиентов банка:

1. Среднее арифметическое,
2. Стандартное отклонение,
3. Минимум,
4. Персентиль 25,
5. Персентиль 50,
6. Персентиль 75,
